Crear Base de Datos

Para crear una base de datos, use el comando

use Videojuegos

Eliminar Coleccion

Para eliminar una coleccion podemos realizarla de la siguiente forma:

db.consola.drop()

Crear Coleccion

Para crear una coleccion utilizamos El método createCollection

db.createCollection('consola');

Eliminar Base de Datos

Para eliminar una base de datos utlizamos el método :

dropDatabase();
